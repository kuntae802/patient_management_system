# Story 1.2: 디자인 시스템 토큰 · 전역 셸 골격

Status: done

<!-- Note: Validation is optional. Run validate-create-story for quality check before dev-story. -->

## Story

As a 프론트엔드 개발자,
I want shadcn/ui on Tailwind 4 위에 브랜드 토큰과 AppShell(사이드바+탑바) 골격, 공통 상태·접근성 기본을 세우기를,
so that 모든 역할 화면이 일관된 시각·인터랙션·접근성 토대를 상속한다.

## Acceptance Criteria

> 출처: [epics.md#Story-1.2]. BDD 원문 그대로. 검증은 각 AC의 **Then/And**를 만족해야 한다.

1. **브랜드 토큰 + WCAG AA (UX-DR1)**
   **Given** Tailwind 4 + shadcn/ui가 설치된 상태에서
   **When** 브랜드 토큰(액센트 teal · 5상태 기능색 · 중립 램프)을 정의하면
   **Then** 모든 색 조합이 WCAG 2.1 AA(본문 ≥4.5:1, 큰 텍스트/UI ≥3:1)를 충족하고, teal은 액션/브랜드 전용이며 상태색으로 쓰이지 않는다

2. **Pretendard 셀프호스팅 + 타입 스케일 (UX-DR2)**
   **Given** 폰트 자산이 번들된 상태에서
   **When** Pretendard 자체 호스팅 + 윈도우 우선 폴백 스택을 적용하면
   **Then** @font-face/외부 네트워크 없이 OS 무관 동일 렌더가 되고, 타입 스케일(page-title/section/body/caption)과 `tabular-nums`가 적용된다

3. **형태·간격·음영 비의존 인코딩 (UX-DR3)**
   **Given** 디자인 시스템이 로드된 상태에서
   **When** 코너 반경 스케일 · big-seams-dense-interiors 스페이싱 · 음영 비의존 인코딩 규칙을 적용하면
   **Then** 중요 신호는 색 채움+테두리+굵기로 표현되고 box-shadow에 의미를 싣지 않는다

4. **AppShell 골격 (UX-DR4)**
   **Given** 인증·권한이 아직 없는 정적 상태에서
   **When** AppShell(좌 사이드바 240/60px + 상단 탑바 52px)을 렌더하면
   **Then** 셸 골격(병원명·시계·벨·아바타 슬롯)이 정상 표시되고, 역할 내비 항목은 권한 연동 자리(placeholder)로 준비된다

5. **공통 상태 + 접근성 기본 (UX-DR18 · UX-DR20)**
   **Given** 공통 UI 상태가 발생할 때
   **When** 로딩=스켈레톤(스피너 금지) · 빈 상태 · 토스트 · `:focus-visible` 포커스 링을 구현하면
   **Then** 네이티브 시맨틱 요소(`<button>`/`<input>`/`<a>`)와 포커스 링이 일관 동작한다

### 🔤 타이포그래피 (UX-DR2)

**Pretendard 자체 호스팅(번들)** — `@font-face`/외부 네트워크 없이 OS 무관 동일 렌더(윈도우·맥·리눅스). `next/font/local`이 정답: 빌트인 셀프호스팅 + 외부 요청 제거 + 레이아웃 시프트 0. **`next/font/google`(Geist) 제거** — 빌드타임 다운로드라 "무네트워크"에 어긋나고, 타깃 폰트가 아님.

```ts
// layout.tsx
import localFont from 'next/font/local'
const pretendard = localFont({
  src: './fonts/PretendardVariable.woff2',  // 리포에 커밋된 자산
  weight: '45 920',          // ⚠ 가변 범위 필수 — 미지정 시 WebKit 가중치 깨짐
  display: 'swap',
  variable: '--font-pretendard',
})
// <html lang="ko" className={pretendard.variable}> ... <body className="font-sans">
```

**폴백 스택(윈도우 우선, `@theme --font-sans`):**
```
var(--font-pretendard), 'Pretendard Variable', Pretendard, 'Malgun Gothic', '맑은 고딕', 'Apple SD Gothic Neo', -apple-system, system-ui, sans-serif
```

**타입 스케일(역할):** page-title 20/600(letter-spacing -0.02em) · section 13/600(muted) · body 13–14/400(lh 1.5, 임상 작성 본문 14) · caption 12/400(muted). **숫자(대기번호·시각·카운트·금액·주민번호)는 `tabular-nums`** 로 세로 정렬. 단축키 힌트는 `Ctrl`(주 타깃 윈도우, `⌘` 금지).

**법정 서식 serif 예외:** `'Batang', 바탕, serif` — 인쇄 법정 양식(진료비 계산서·영수증·세부산정내역서) 전용. **화면 UI 미적용**(Epic 7에서 인쇄 레이아웃에 사용). 지금은 토큰 자리만.
